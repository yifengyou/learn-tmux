# 基本概念



## 前缀快捷键

**tmux 中所有的快捷键都需要和前缀快捷键 ctrl + b  来组合使用**

* tmux环境下，左右快捷键都必须经过前缀快捷键发起，也就是说任意快捷键至少3个按键触发
